
## Steps to creat C# .net program

- Creat a new folder.
- Open the terminal where you want to creat console app.
- Write the command `dotnet new console` for creating console app.
- Open Program.cs file and write the program you want.
- Now go to terminal at same location and write the command `dotnet run` to run the console app.
- In the console you will see the output you written in the Program.cs file.
